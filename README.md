### Framework

| Framework           | Purpose                                                  |
|---------------------|----------------------------------------------------------|
| **Photos**           | 커스텀 앨범 구현                     |
| **CoreLocation**     | 위치 권한 요청 및 위치 기반 기능 구현                     |

| **Firebase Crashlytics** | 실시간 크래시 리포팅 및 디버깅 지원               | `11.11.0`    |
